## Usage
- A primary szín a fő CTA gombok, linkek, aktív állapotok és kiemelt elemek színe.
- A secondary szín használható kiegészítő akciókhoz, highlightokhoz, seat map státuszokhoz és kisebb UI hangsúlyokhoz.
- A light palette legyen az alap háttér- és felületkészlet.
- A dark palette főleg szövegre, ikonokra, headingekre és erős kontrasztú elemekre szolgáljon.
- A semantic színek kizárólag státuszokhoz, visszajelzésekhez, badge-ekhez és értesítésekhez használhatók.
- A CTA-k és fontos interakciók ne használjanak túl sok különböző hangsúlyszínt egyszerre.
- Ellenőrizni kell a WCAG 2.1 AA kontrasztot különösen gombok, címkék és inputok esetén.
- A javasolt arány: 70% neutral/light, 20% primary/dark structure, 10% accent/semantic emphasis.
