## Usage
- A **primary** (Royal Blue) szín a fő CTA gombok, linkek, aktív állapotok és a kijelölt ("LockedByMe") székek színe.
- A **secondary** (Amber) szín használható kiemelt akciókhoz (pl. "Proceed to Checkout" gomb), figyelemfelhívó elemekhez.
- A **light** palette a tiszta, minimalista háttérkészlet. A `#F9FAFB` az alap háttér, a `#E5E7EB` a szabad székek színe, a `#9CA3AF` pedig a foglalt ("Booked") székeké.
- A **dark** palette főleg szövegre, ikonokra, headingekre és erős kontrasztú elemekre szolgál.
- A **semantic** színek (Success/Emerald, Warning/Orange, Danger/Red) a VIP/Standard/Olcsó árkategóriák színezéséhez a seat mapen, illetve státuszüzenetekhez használhatók.
- A javasolt arány: 70% neutral/light, 20% primary/dark structure, 10% accent/semantic emphasis.
