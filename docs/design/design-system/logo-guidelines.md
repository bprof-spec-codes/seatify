# Seatify Logo Guidelines

## 1. Logo Architecture
The Seatify logo consists of two core elements:
- **The Icon:** A minimalist, stylized symbol representing a chair/seat, nodding to the platform's core booking feature.
- **The Wordmark:** The text "Seatify" written in a clean, sans-serif typography, formatted in Title Case.

The primary configuration is the **Horizontal Layout**: the icon sits on the left, followed by the wordmark on the right.

## 2. Logo Variations
- **Primary Logo:** Royal Blue (`#3B82F6`) icon with Dark (`#111827`) text. This is the default logo used on all light backgrounds (app header, landing page).
- **Monochrome Dark:** Solid dark gray/black logo. Used for printed materials or highly simplified single-color designs.
- **Monochrome Light (Inverted):** Solid white logo. Strictly used when the logo is placed on dark backgrounds (e.g., dark blue banners or photography).
- **Icon Only (Favicon/Avatar):** The chair icon standalone. Used for profile avatars, browser favicons, and highly restricted mobile UI spaces.

## 3. Clear Space
Always maintain sufficient clear space around the logo to prevent it from blending with other UI elements.
- The minimum clear space is equal to **50% of the icon's height (0.5x)**. No other graphics, text, or edge boundaries should enter this safe zone.

## 4. Minimum Size
To maintain legibility across digital interfaces, adhere to these minimum sizes:
- **Full Logo (Icon + Text):** Minimum 100px in width.
- **Icon Only:** Minimum 24x24px (for UI icons) or 16x16px (for Favicons).

## 5. Usage Restrictions (Don'ts)
To ensure brand consistency, the following alterations are **STRICTLY PROHIBITED**:
1. **Do not distort:** Never stretch, squash, or warp the logo. Always scale proportionally.
2. **Do not change colors:** Do not use colors outside the official brand palette (e.g., no red or purple logos).
3. **Do not rotate:** The logo must always sit perfectly horizontal.
4. **Do not place on busy backgrounds:** Avoid placing the logo on cluttered photographs without sufficient contrast. Use solid backgrounds or dark overlays.
5. **Do not alter proportions:** Do not resize the icon relative to the wordmark.

## 6. App Navigation Behavior
Following global UI/UX standards, the primary logo is placed in the top-left corner of the application header. **It must always act as an active, clickable link that returns the user to the Homepage/Landing Page.**
